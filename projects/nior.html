NIOR


Description

Nior was the second game done over the span of a week for a Game Engine Lab class taken at UCI. Lacking sound, the game was focused on implementing at least 5 rules from Jesper Juul's Classic Game Model and a minimum of 5 game mechanics.

In Nior, you journey as a silent knight who aims to reach the end of a never-ending 2D path. You will come across far-reaching platforms and cursed skeletons that have risen from the dead, but it is nothing the knight won't be able to handle, right?

Installation Instructions

Download the game folder here: nior_game.zip
Extract the downloaded .zip folder and run the .exe file.
