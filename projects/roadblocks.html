ROADBLOCKS


Description

Roadblock was the first game done over the span of a week for a Game Engine Lab class taken at UCI. Lacking sound, the game was focused on implementing at least 5 rules from Jesper Juul's Classic Game Model and is required to have at least 1 game mechanic.

Roadblock is a top-down, flash-like game where you will have to maneuver a simple maze without touching the red stars. The game was designed to give players a sense of dread and fear, forcing you to move through the maze carefully. You will be rewarded for taking your time.

Installation Instructions

Download the game folder here: roadblocks_game.zip
Extract the downloaded .zip folder and run the .exe file.
