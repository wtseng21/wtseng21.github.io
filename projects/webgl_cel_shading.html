WEBGL CEL SHADING


Description

WebGL Cel Shading was a final project that we proposed to do for our Advanced 3D Computer Graphics course. We decided to put our own twist on the typical cel shading implementation by attempting to use the sobel operator to detect the edges for outlining. However, we ended up outlining all edges including the ones generated from cel shading and experienced significant aliasing issues.

Features

• Ability to change the object model
• Change the resolution of the canvas
• Rotate the light and object around the center
• Change the number of cel bands from cel shading
• Alter the diffuse color and outline color of the object
• Enable/Disable cel shading and outlining
• Adjust the strength of the light
